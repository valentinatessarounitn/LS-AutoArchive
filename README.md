# AutoArchive
Limesurvey plugin to identifies, archives, and deletes surveys older than a specified number of months

## Installation

This version is for LimeSurvey 6.

### Via GIT
- Go to your LimeSurvey Directory
- Clone in plugins/AutoArchive directory : `git clone https://github.com/valentinatessarounitn/LS-AutoArchive.git AutoArchive`

### Via ZIP dowload
- Download <https://github.com/valentinatessarounitn/LS-AutoArchive/archive/refs/heads/main.zip>  
- Extract `unzip LS-AutoArchive-main.zip` as `AutoArchive`
- Move the directory `AutoArchive` to  plugins/ directory inside LimeSurvey
